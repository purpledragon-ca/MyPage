## Install MoveIt and ros2_control Packages (ROS 2 Jazzy)

In your ROS 2 workspace `src` directory:

```bash
sudo apt update
sudo apt install \
  ros-jazzy-moveit \
  ros-jazzy-ros2-control \
  ros-jazzy-ros2-controllers \
  ros-jazzy-gripper-controllers \
  ros-jazzy-topic-based-ros2-control
```

## MoveIt Setup Assistant
```bash
ros2 launch moveit_setup_assistant setup_assistant.launch.py
```

## Fix joint_limits.yaml Bug Generated by MoveIt
    Make sure all numbers are float, not integer type.

## ros2_control Configuration 
In the SO-ARM URDF, add or modify the ros2_control block to use topic_based_ros2_control and match the Isaac Sim topics:
```xml
<hardware>
    <!-- Use a topic-based control system for simulation -->
    <!-- plugin>mock_components/GenericSystem</plugin -->
    <plugin>topic_based_ros2_control/TopicBasedSystem</plugin>

    <!-- Topics that match Isaac Sim Action Graph -->
    <param name="joint_states_topic">/isaac_joint_states</param>
    <param name="joint_commands_topic">/isaac_joint_command</param>
</hardware>
  ```
## Add action for Controller adjustment moveit_controllers.yaml. 
```yaml
action_ns: follow_joint_trajectory
default: true
```
